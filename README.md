# elfvideo
Script to generate TwitchPlaysPokemon Viet Crystal (and others) timelapse compilation video


## Usage

Requires:

* Python 3.4+
* PyCairo

Run:

        python3 elfvideo.py ./input_dir/ ./output_dir/
